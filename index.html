Lettershop & Fulfillment Tabs

  • Inkjet Proofs- does selection match IJ proof box
  • Is the following correct and does it match Postage and Sample
    • Piece Size
    • Font/Size/Color
    • Barcode
    • OEL
    • T&P
    • Seq #
  • Stamp/Permit (Stamp Order Email Needed?)
    • Is it a retail flat? In Fulfillment section?
    • Is it a retail letter over 1oz? In Fulfillment section?
  • # of Inserts Correct entered in Match or Insert description
  • Special Proofing Requirements
  • All necessary services codes added?

Print Tab

  • Include File Names
  • Bleeds or No Bleeds
  • Color Match (PMS or Physical)
  • Final Proof (Client / PM / Hardcopy)
  • Folding
  • Set quantity printing
  • Prepress Needed (15 Min or Less)

Foreign Tab

  • Set Up Foreign
  • “Same As Domestic”
  • Additional work needed
    • Envelopes needed
    • Labels needed; consider formatting
  • Postage
  • Apply stamps
  • Foreign weight different

Inventory and Warehouse Tabs

  • Inventory received?
    • Verify quantities received vs mail quantity
    • If short, include instructions or contact printer/client
  • Direction for handling of residuals
    • If return, is the shipping address in ship description? Street address, not P.O. Box?
    • Have you input how many pieces, folded or flat, stuffed, addressed in service code?
  • Deliver to NDC/SCF

Sample Review

  • Consolidated Sample: is the following correct and does it match other information?
    • postage affix
    • weight
    • Seq #-Highlighted
    • T&P
    • Are the pieces numbered?
  • Fold Samples
  • Foreign Sample
    • "Same As Domestic"-matches consolidated; change postage?
    • Processing different-include envelope and label layout
    • Weight different-include on sample
  • Tabs drawn on Self Mailers/ Booklet sample
  • Letter-sized piece meets size & aspect ratio
  • Mail Panel Approved
    • Used mail panel overlay
    • At least 2x4
    • Has return address on front/back
    • Return address matches NP# & CRID address
